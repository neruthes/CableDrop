# CableDrop

## Abstract

Early WIP.

Tool for sharing files like AirDrop.

## Usage

### Receiving

Start `cabledropd` as a daemon with desktop GUI session.

### Sending

Execute `cabledrop FILENAME` to send a file by selecting a receiver from the list.



## Dependencies

- avahi
- daemonize
- nodejs
- python3
